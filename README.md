# Voice-Assessment-Platform

Interactive voice assessment suite tracking customer interactions with webhooks and reports.

![Python](https://img.shields.io/badge/Python-3.11-blue?logo=python)
![FastAPI](https://img.shields.io/badge/FastAPI-0.110-green?logo=fastapi)
![React](https://img.shields.io/badge/React-18.0-blue?logo=react)

## Overview
Voice-Assessment-Platform conducts user evaluations via audio dialogue. It logs grades, saves sessions, and fires external webhooks upon assessment completion.

## Features
- **Interactive Evaluations** — Conducts verbal sessions, storing assessment paths.
- **Webhook Gateway** — Integrates reports with external platforms on completion.
- **Dashboard Workspace** — Interfaces for admins to manage reports and grades.

## Tech Stack
| Layer | Technology |
|-------|-----------|
| Backend | FastAPI, SQLAlchemy, Alembic, SQLite |
| Frontend | React, Tailwind CSS |
